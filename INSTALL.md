# HydroInsure — Installation

1. Download `HydroInsure.zip`
2. Extract to any folder (e.g., `C:\HydroInsure`)
3. Double-click `HydroInsure.exe`
4. On first launch: go to **Data Setup** tab, click "Run Data Pipeline"

Note: ERA5 fetch requires a free account at [cds.climate.copernicus.eu](https://cds.climate.copernicus.eu).

Enter your API key in **Settings** (gear icon) before fetching ERA5.
Once data shows all green: go to **Hydrology** tab and run the model.

No Python installation required.
